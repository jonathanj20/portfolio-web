Boda web

Descripción

Aplicación web para una boda que incluye invitaciones personalizadas para cada invitado, una galería con fotos de los novios, ubicaciones y confirmación de asistencia por whatsapp.
Los datos que contienen las invitaciones personalizadas son el nombre del invitado, el número de adultos que puede llevar el invitado a la boda y el número de niños en caso de que al invitado se le haya permitido llevar niños a la boda, para que el usuario pueda ver su invitación debe de ingresar su nombre en el campo de texto al inicio de la página.

Aprendizaje

En este proyecto aprendí a:

  • Resolver problemas utilizando librerías, por ejemplo, para crear un slider en la galería de imagenes usé la librería swiper para agilizar y facilitar su desarrollo.
  • Conectar una base datos y ejecutar consultas con PHP.
  • Integrar nuevas fuentes de letra y colores personalizados en Tailwind.
  • Utilizar la API de WhatsApp para redireccionar al usuario a los chats de los novios.

El Paceño

Descripción

Página web que muestra información acerca del negocio de comida "El Paceño". Contiene una sección que muestra la promoción que se encuentra disponible en el momento, un menú en donde se pueden ver todas las comidas y bebidas con su respectiva descripción y precio y una sección para ver la ubicación y horario.

Aprendizaje

En este proyecto aprendí a:

  • Implementar la librería react router y usar sus componentes para la creación de nuevas rutas dentro de una aplicación.
